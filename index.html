GitHub Integration RECOMMENDED

Auto-deploy setiap kali Anda push code! Best practice untuk project jangka panjang.

1
Upload project ke GitHub repository
2
Buka vercel.com/new
3
Klik "Import Git Repository"
4
Pilih repository Anda dan klik "Import"
5
Klik "Deploy" - Auto-deploy enabled! üöÄ

Vercel CLI

Deploy langsung dari terminal - untuk developer yang suka command line!

1
Install Vercel CLI
npm install -g vercel
2
Masuk ke folder project
cd my-project
3
Deploy dengan satu command
vercel
4
Follow instruksi di terminal (login, pilih settings)
5
Deployment URL akan muncul di terminal! üéâ
